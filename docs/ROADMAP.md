# QuantumStateObjects Experimenters Roadmap

## Purpose
Define the bounded object model, genome loading, memory, cognition, communication, freeze, policy, and serialization layers required by Experimenter QSOs.

## Phases
1. Contracts and immutable state.
2. Lifecycle and bounded memory.
3. Communication and freeze behavior.
4. Policy enforcement and serialization.
5. Integration, assurance, and release review.

## Rule
Every scaffold file is a development specification, not a capability claim. Implementations must fail closed and require exact-head evidence before consumption by QSO-FABRIC.
